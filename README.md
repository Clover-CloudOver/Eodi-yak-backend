## 주요 기능
> **약 재고 검색**
- 사용자 위치 1km 이내의 약국의 의약품 재고 정보 제공

> **약 예약 기능 (부분 구현)**
- 사용자가 원하는 의약품 예약

> **약 재입고 알림 기능 (부분 구현)**
- 특정 의약품 재입고 알림 신청
